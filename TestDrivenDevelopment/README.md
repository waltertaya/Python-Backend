# Test Driven Development (TDD)

## Assert methods in the unittest

- `assertEqual(a, b) :: a == b`
- `assertNotEqual(a, b) :: a != b`
- `assertTrue(x)`
- `assertFalse(x)`
- `assertIs(a, b) :: a is b`
- `assertIsNot(a, b)`
- `assertIsNone(x)`
- `assertIsNotNone(x)`
- `assertIn(a, b) :: a in b`
- `assertNotIn(a, b)`
- `assertIsInstance(a, b)`
- `assertNotIsInstance(a, b)`
- `assertRaises() -> Uses context manager`
- `assertWarns()`
