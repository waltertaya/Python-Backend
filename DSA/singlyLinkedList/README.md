# Singly Linked List

- Chain of nodes
- Example
```bash
node---->node---->node----->node
```

- Node contain two fields
    1. Data field
    2. Link/reference (to the next node)
- Last node store the link to null value
- The first node is called `head`

### Operations in Singly linked list

- Inserting/Adding
    1. Insert new node as the head node
- Removing/Deleting
- Traversal
    1.
